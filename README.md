# ESSLLI 2017 Embeddings and Deep Learning

This repository contains the example code for the ESSLLI 2017 lecture on Embeddings and Deep Learning by Hinrich Schütze and Martin Schmitt.

The neural network implementation is based on [this code](https://github.com/Shawn1993/cnn-text-classification-pytorch).

See the [official course website](http://cis.lmu.de/esslli2017) for further information as well as links to the datasets used during the course.
